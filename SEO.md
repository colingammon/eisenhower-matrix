# SEO Optimization Guide

This document outlines the SEO optimizations implemented for Eisenhower Matrix on the custom domain `eisenhower-matrix.online`.

## What's Been Done

### 1. **Meta Tags** ✅
- Optimized page title: "Eisenhower Matrix - Free Task Prioritization App"
- Comprehensive meta description (160 characters)
- Keywords targeting: productivity, task management, prioritization
- Canonical URL pointing to your custom domain
- Robots meta tag allowing indexing

### 2. **Open Graph Tags** ✅
- Configured for Facebook, LinkedIn, WhatsApp sharing
- Dynamic title, description, and image
- Social media preview shows app purpose

### 3. **Twitter Card** ✅
- Summary card with large image
- Proper Twitter metadata for rich previews

### 4. **robots.txt** ✅
- Located at: `/robots.txt`
- Allows all search engines to crawl and index
- References sitemap.xml
- Crawl-delay: 1 second

### 5. **sitemap.xml** ✅
- Located at: `/sitemap.xml`
- Lists main URL with priority 1.0
- Weekly change frequency
- Helps Google discover content faster

### 6. **Structured Data (JSON-LD)** ✅
- Application schema markup
- Tells Google: type, description, category, price (free!)
- Lists key features
- Better search result appearance (rich snippets)

### 7. **Apple Touch Icon** ✅
- Favicon for iOS home screen pinning
- Shows custom app icon when bookmarked
